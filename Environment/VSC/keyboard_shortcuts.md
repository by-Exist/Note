# **[ VSC ]**

# **[ File management ]**

| 단축키                     | 동작                                     |
| -------------------------- | ---------------------------------------- |
| Ctrl + N                   | 새 파일 작성하기                         |
| Ctrl + S                   | 파일 저장                                |
| Ctrl + Shift + S           | 다른 이름으로 파일 저장                  |
| Ctrl + W                   | 파일 닫기                                |
| Ctrl + Shift + T           | 마지막으로 닫은 파일 다시 열기           |
| Ctrl + (Tab / Shift + Tab) | 편집중인 이전 / 다음 파일 선택하기       |
| Ctrl + PgUp / PgDn         | 편집중인 이전 / 다음 파일 선택하기       |
| Ctrl + Shift + PgUp / PgDn | 편집중인 파일의 순서 변경하기            |
| Ctrl + K, P                | 해당 파일의 절대 경로 복사하기           |
| Ctrl + P                   | 파일 검색창 열기                         |
| └ :                        | 현재 파일의 라인 번호를 입력하여 이동    |
| └ @                        | 현재 파일의 심볼 중 하나를 선택하여 이동 |
| Ctrl + K, R                | 해당 파일이 있는 폴더를 탐색기로 열기    |

# **[ Basic editing ]**

| 단축키               | 동작                                 |
| -------------------- | ------------------------------------ |
| Alt + Z              | 자동 줄바꿈 설정 / 해제              |
| Ctrl + Z             | 작업 취소                            |
| Ctrl + X             | (드래그 X) 라인 잘라내기             |
| Ctrl + C             | (드래그 X) 라인 복사                 |
| Ctrl + Shift + K     | 라인 삭제                            |
| Alt + ↑ / ↓          | 라인 올리기 / 내리기                 |
| Ctrl + Enter         | 아래에 빈줄 추가                     |
| Ctrl + Shift + Enter | 위에 빈줄 추가                       |
| Home / End           | 라인의 처음으로 / 끝으로 이동        |
| Ctrl + Home / End    | 파일의 처음으로 / 끝으로 이동        |
| Ctrl + ] / [         | 들여쓰기 추가 / 제거                 |
| Ctrl + ↑ / ↓         | 줄 단위로 스크롤 올리기 / 내리기     |
| Alt + PgUp / PgDn    | 페이지 단위로 스크롤 올리기 / 내리기 |
| Ctrl + /             | 라인 주석 설정 / 해제                |

# **[ Search and replace ]**

| 단축키                      | 동작                                                |
| --------------------------- | --------------------------------------------------- |
| Ctrl + F                    | 찾기 열기                                           |
| Ctrl + H                    | 바꾸기 열기                                         |

| (검색 중) Ctrl + K Ctrl + D | 해당 항목 선택하지 않고 다음 항목 선택하기          |
| (검색 중) Ctrl + Shift + L  | 모든 항목 선택하기                                  |
| (검색 중) Alt + Enter       | 모든 항목 선택하기                                  |
